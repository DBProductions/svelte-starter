# svelte-starter

Starter repository for the magical disappearing UI framework.  
[Svelte Guide](https://svelte.technology/guide)  

## Installation

    $ npm install -g rollup
    $ npm install

## Compile

    $ npm run compile
    $ rollup -c --watch

## Browse

Open `public/index.html` in your browser to see the result.
